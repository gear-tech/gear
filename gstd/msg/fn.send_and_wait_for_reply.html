Send a message and wait for reply.

This function works similarly to send_bytes_and_wait_for_reply, with one difference - it takes the structure in, then encodes it and sends it in bytes. When receiving the message, it decodes the bytes. So the input should be SCALE codeс encodable, output - decodable (https://docs.substrate.io/v3/advanced/scale-codec/). The program will be interrupted (waiting for a reply) if an .await has been called on the CodecMessageFuture object returned by the function.
