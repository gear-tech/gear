Function gstd::exec::permute

source · 
pub fn permute(data: [u64; 12]) -> Result<[u64; 12], Error>
Expand description

Perform input data transformation and return data in the same format. Primarily designed for the Poseidon permutation of Goldilocks field arrays, represented as fixed-size arrays of u64 type - the key part of the Poseidon hash function. For the Goldilocks field the array size is equal to 12.

data is the permutation input.

§Examples

use core::array;
use gcore::exec;

#[no_mangle]
extern "C" fn handle() {
    let data: [u64; 12] = array::from_fn(|i| i as u64 + 1);
    let hash_out = exec::permute(data).expect("Error in random");
}
