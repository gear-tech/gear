Expand description

Signer representation that provides access to gear API. Implements low-level methods such as run_tx and force_batch. Other higher-level calls are provided by Signer::storage, Signer::calls, Signer::rpc.

Run transaction.

This function allows us to execute any transactions in gear.

ยงYou may not need this.

Read the docs of Signer to checkout the wrappred transactions, we need this function only when we want to execute a transaction which has not been wrapped in gsdk.

ยงExample
โ
use gsdk::{
  Api,
  Signer,
  metadata::calls::BalancesCall,
  Value,
};

let api = Api::new(None).await?;
let signer = Signer::new(api, "//Alice", None).await?;

{
    let args = vec![
        Value::unnamed_variant("Id", [Value::from_bytes(dest.into())]),
        Value::u128(value),
    ];
    let in_block = signer.run_tx(BalancesCall::TransferKeepAlive, args).await?;
}
