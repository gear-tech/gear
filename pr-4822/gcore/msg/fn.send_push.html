Function send_push

Source 
pub fn send_push(handle: MessageHandle, payload: &[u8]) -> Result<()>
Expand description

Push a payload part of the message to be sent in parts.

Gear allows programs to work with messages in parts. This function adds a payload part to the message specified by the message handle.

The first argument is the message handle MessageHandle that specifies a particular message built in parts. The second argument is the payload buffer.

ยงExamples

use gcore::msg;

#[unsafe(no_mangle)]
extern "C" fn handle() {
    let msg_handle = msg::send_init().expect("Unable to init");
    msg::send_push(msg_handle, b"Hello,").expect("Unable to push");
    msg::send_push(msg_handle, b" world!").expect("Unable to push");
    msg::send_commit(msg_handle, msg::source(), 42).expect("Unable to commit");
}
