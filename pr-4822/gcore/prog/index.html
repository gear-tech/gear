API for creating programs from Gear programs.

Once a Wasm code has been uploaded to the chain, the Gear program can create a new program from the code blob using its identifier.

To create a new program, you are to provide the code identifier obtained after running the upload_code extrinsic, unique salt (arbitrary data needed to instantiate several programs from one code blob), and the init message that consists at least from a payload and value.
